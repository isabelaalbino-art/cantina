# 📖 Sobre o Projeto

O **Cardápio Digital da Cantina SENAI "Edward Sávio"** foi desenvolvido com o objetivo de modernizar o atendimento da cantina escolar, reduzindo filas durante os intervalos e facilitando o acesso ao cardápio.

O sistema permite que os alunos consultem os produtos disponíveis, realizem pedidos online para retirada presencial e visualizem o cardápio das refeições do dia.

Além disso, possui uma área administrativa onde os responsáveis pela cantina podem atualizar os produtos, preços e disponibilidade do cardápio.

---

# 🎯 Objetivos

- Reduzir filas na cantina;
- Diminuir dúvidas sobre preços e produtos;
- Disponibilizar o cardápio atualizado;
- Facilitar pedidos online;
- Melhorar a organização da cantina;
- Centralizar o gerenciamento do cardápio.

# 🖥️ Telas do Sistema

O projeto possui quatro páginas principais:

## 1. Página Inicial

- Visualização do cardápio
- Categorias dos produtos
- Cardápio das refeições
- Botão para realizar pedido

---

## 2. Página de Compra

Permite:

- selecionar produtos;
- alterar quantidades;
- visualizar o resumo do pedido;
- escolher forma de pagamento;
- finalizar o pedido.

---

## 3. Login do Administrador

Tela destinada apenas aos funcionários da cantina.

Possui:

- login;
- senha;
- validação dos campos;
- redirecionamento para área administrativa.

---

## 4. Área Administrativa

Permite:

- pesquisar produtos;
- editar itens;
- excluir produtos;
- alterar preços;
- atualizar refeições do dia.
